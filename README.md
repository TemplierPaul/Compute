# Compute
Bash tool to start multiple Slurm jobs easily

## Install
### Clone
```bash
git clone git@github.com:TemplierPaul/Compute.git
chmod a+x Compute/run.sh
```

### Alias
Add to `.bash_aliases`: 
```
alias run="<path_to_run.sh>"
```

### Edit paths in files

## Run
- Edit `joblist.txt`
- Run all configs: 
```
run all
```
- Run specific configs:
```
run cpu
```
